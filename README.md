## 🎨 UI/UX Architecture

The portfolio has been recently overhauled to deliver a premium, cinematic experience:

*   **Neon-Noir Aesthetic**: A deep dark mode (`#0A0A0A`) foundation paired with a subtle CSS noise/grain overlay to create a lo-fi, cinematic texture.
*   **Glassmorphism**: Implementation of frosted glass effects (`backdrop-filter: blur`) on strategy and project cards for a modern, lightweight feel.
*   **Fluid Motion**: Integration of **AOS (Animate On Scroll)** for buttery smooth fade-up and fade-in transitions, enhancing the scrollytelling experience.
*   **Bilingual Integration**: Seamless, dynamic switching between **English** and **Myanmar (Burmese)** languages across the entire interface, utilizing `localStorage` for user preference retention.
*   **Typography**: Utilization of `Space Grotesk` and `Orbitron` for a bold, technical, yet artistic typographic hierarchy.

## 🛠 Tech Stack

The portfolio is built with a focus on lightweight performance and custom styling:

*   **Core**: `HTML5`, `CSS3`, `Vanilla JavaScript`
*   **Animations**: `AOS (Animate On Scroll)`
*   **Typography**: Google Fonts (`Space Grotesk`, `Orbitron`)
*   **Form Handling**: `Formspree` API integration
*   **Deployment**: `Vercel`

## 🚀 Getting Started

To run this portfolio locally for development or review:

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/theimhtikesoe/Rhyzoe-Portfolio.git
    ```
2.  **Navigate to the directory**:
    ```bash
    cd Rhyzoe-Portfolio
    ```
3.  **Launch the application**:
    Simply open the `index.html` file in your preferred modern web browser. No build steps or package installations are required.
